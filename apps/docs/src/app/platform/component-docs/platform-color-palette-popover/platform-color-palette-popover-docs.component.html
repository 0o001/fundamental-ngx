Simple Color Palette Popover The default color palette popover Complex Color Palette Popover The color palette popover can be expanded with the following properties:
  • defaultColor="color" input a css color that sets the default color. Used with showDefaultColor.
  • [showDefaultColor]="true" shows a button that allows user to choose the default color. used with defaultColor.
  • [showMoreColors]="true" defines whether the user can choose a custom color from a color-picker.
  • [defaultColor]="true" defines whether the user can see the last used colors in the bottom of the component
