Color Palette Popover
The Color Palette Popover provides users with a range of predefined colors. The colors are fixed and do not change with the theme.
To define the colors, use the fdp-color-palette-item component as a child inside the fdp-color-palette-popover.

To open the popover, call the method showPopover(), on the popover component while passing in the element where the popover is shown at.
