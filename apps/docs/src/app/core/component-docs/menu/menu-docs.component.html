Menu Menu with Addon This is an additional container that can be used for an icon before or after the menu item text. This can be achieved with the directives fd-menu-item-addon-before and fd-menu-item-addon-after Menu specified as a configuration Menu with all its attributes specified in a configuration rather than writing in template. Menu with Optional Separator Each menu item can have an optional separator shown as a line beneath the item. Menu with Different Sizes Menu can be in compact('desktop') size or cozy size(tablet is default and 'mobile' is for mobile). Menu with Submenu Menu can be opened with a submenu for further navigation and grouping. Contextual Menu Menu is generally opened with a popover. Keyboard Support Keyboard support can be added on every component which uses the MenuComponent. It simulates click event on Enter or Space key down. It is also possible to override functions for a variety of situations. For example, when the focus should escape the element. Use the up and down arrows to test the component below. To enable focusing on non-focusable elements, change value of tabIndex to 0.
