Selector: <fd-date-picker>

The datetime picker component is an opinionated composition of the fd-popover and fd-calendar components to accomplish the UI pattern for picking a date.

Supports Angular Forms.

<fd-date-picker [(ngModel)]="date"></fd-date-picker>
