Inputs

appendToWizard

appendToWizard: boolean = true

Whether or not to append the step to the wizard. If false, each step will be displayed on a different page. Default is true.

ariaLabel

ariaLabel: string = 'Wizard'

adding aria label to the component

contentHeight

contentHeight: Nullable<null | string>

Custom height to use for the wizard's content pane. By default, this value is calc(100vh - 144px), where 144px is the combined height of the shellbar, wizard header and wizard footer.

displaySummaryStep

displaySummaryStep: boolean = false

responsivePaddings

responsivePaddings: boolean = false

Whether or not apply responsive paddings styling.
