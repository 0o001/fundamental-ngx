Alert component is deprecated since version 0.16.0 Message Strip component should be used instead.

Methods

dismiss

  • dismiss(reason?, manualDismiss?): void

  • Dismisses the alert. If the alert was generated via the AlertService, it is removed from the DOM. Otherwise, it sets the display value to none. Fires the onDismiss event.

    Parameters

    • Optional reason: any

      Data to pass back to the calling component. Only usable if alert is opened using the Service.

    • manualDismiss: boolean = false

      Set to true to skip the dismiss animation.

    Returns void
