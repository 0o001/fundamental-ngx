Inputs

collapseOverflow

collapseOverflow: boolean = true

Whether to move tabs overflowing in the tab bar to the dropdown

collapsibleTabs

collapsibleTabs: boolean = false

Whether to enable collapsing expanded tab on expanded tab click

expandOverflowText

expandOverflowText: string = 'More'

Text visible in expand overflow trigger

maxContentHeight

maxContentHeight: string = '100%'

Maximum height of the content

maxVisibleTabs

maxVisibleTabs: Nullable<number> = null

Limits the maximum number of tabs visible in the tab bar in collapseOverflow mode. Other tabs will be moved to the collapsed tabs dropdown

mode

mode: TabModes

Whether user wants to use tab component in certain mode. Modes available: 'icon-only' | 'process' | 'filter'

size

size: TabSizes = 'm'

Size of tab, it's mostly about adding spacing on tab container, available sizes 's' | 'm' | 'l' | 'xl' | 'xxl'

stackContent

stackContent: boolean = false

Whether to open tab content one under another without collapsing

Outputs

selectedTabChange

selectedTabChange: EventEmitter<TabPanelComponent> = ...

Event emitted when the selected panel changes.
