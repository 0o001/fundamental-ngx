Selector: <fd-breadcrumb>

Breadcrumb parent wrapper directive. Must have breadcrumb item child directives.

<fd-breadcrumb>
    <fd-breadcrumb-item>
        <a fd-link [routerLink]="'#'">Breadcrumb Link</a>
    </fd-breadcrumb-item>
</fd-breadcrumb>

Inputs

Optional compact

compact: boolean

Whenever links wrapped inside overflow should be displayed in compact mode

containerElement

containerElement: HTMLElement

The element to act as the breadcrumb container. When provided, the breadcrumb's responsive collapsing behavior performs better. When not provided, the immediate parent element's width will be used.

reverse

reverse: boolean = false

Whether to append items to the overflow dropdown in reverse order. Default is true.
