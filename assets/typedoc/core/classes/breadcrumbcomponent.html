Selector: <fd-breadcrumb>

Breadcrumb parent wrapper directive. Must have breadcrumb item child directives.

<fd-breadcrumb>
    <fd-breadcrumb-item>
        <a fd-breadcrumb-link [routerLink]="'#'">Breadcrumb Link</a>
    </fd-breadcrumb-item>
</fd-breadcrumb>
