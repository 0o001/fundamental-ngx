getColumnWidthStyle

  • getColumnWidthStyle(column: TableColumn): string

    Get the column width, try in the next order:

    1. Width from map with calculated widths.
    2. Width defined by the user.
    3. Literally auto, means no width set.

    In case there was a change to the width of the column (from the component input or resize event), corresponding value will be treated as higher priority

    Parameters

    • column: TableColumn

    Returns string
