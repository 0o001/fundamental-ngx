Selector: <fd-inline-help>

The component that represents an inline-help. Inline help is used to display help text in a popover, often inline with headers, body text and form labels.

<fd-inline-help [position]="'bottom-left'">
     Lorem ipsum dolor sit amet, consectetur adipiscing.
</fd-inline-help>
